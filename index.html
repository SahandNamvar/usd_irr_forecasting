About the Data Source

This dataset contains comprehensive historical exchange rates for the USD to IRR (Rial), updated and preprocessed daily to ensure accuracy.

  • Data Source: TGJU.org (Tehran Gold and Jewelry Union)
  • Update Frequency: Daily via GitHub Actions
  • Forecasting Model: LSTM (Long Short-Term Memory)
  • Currency Pair: USD / IRR (Rial)
  • Use Case: Aid in financial planning and analysis
Download Dataset (CSV)

About the AI Model

The forecasting engine uses a Long Short-Term Memory (LSTM) neural network trained mainly for financial time-series prediction.

  • Training Data: 14+ years of daily historical rates
  • Input Window: Prior 90 days of pricing
  • Accuracy: Shown under "Forecasted Price Metrics"
  • Limitations: Subject to market volatility and external factors
  • Development: Built with Python, TensorFlow, and deployed via GitHub Actions
